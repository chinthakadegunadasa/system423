# පරිච්ඡේදය 02: අප්පමාද වග්ගය (The Chapter on Vigilance)
### පංච උපාධානස්කන්ධයේ උදය වැය සහ කාල-සමමුහුර්තකරණ පද්ධති (Time-Sync Systems)

#### 2.1 අප්‍රමාදය: පද්ධතිමය නිර්වචනය (System Definition)
ධම්මපදයේ දෙවන වග්ගය වන අප්පමාද වග්ගය, පද්ධති ඉංජිනේරු විද්‍යාත්මකව හඳුනාගත හැක්කේ **"High-Resolution Real-time Monitoring"** ක්‍රියාවලියක් ලෙසයි. මෙහිදී අප්‍රමාදය යනු හුදු අවධානයෙන් සිටීම පමණක් නොව, පංච උපාධානස්කන්ධයේ ගතික ස්වභාවය නිරීක්ෂණය කිරීමයි.

* **උදය වැය (Arising and Passing):** පද්ධතියක දත්ත පැකට්ටුවක් (Data Packet) ජනනය වීම සහ විනාශ වීම අතර පවතින කාල පරාසය නිරීක්ෂණය කිරීම.
* **නිවනට මග:** පද්ධතියේ "ධාරිතාව" (State) ස්ථිර ඒකකයක් ලෙස නොව, අඛණ්ඩව වෙනස් වන දත්ත ප්‍රවාහයක් (Stream) ලෙස හඳුනා ගැනීමෙන් පද්ධතිමය බන්ධනයෙන් (Attachment) නිදහස් වීම.

#### 2.2 පංච උපාධානස්කන්ධයේ ස්නායුක සහ පද්ධතිමය විශ්ලේෂණය

| ස්කන්ධය | පද්ධතිමය භූමිකාව | උදය වැය නිරීක්ෂණය (Observation Logic) |
| :--- | :--- | :--- |
| **රූප** | Hardware Interface | භෞතික සංඥාවල අනිත්‍ය ස්වභාවය හඳුනා ගැනීම. |
| **වේදනා** | Signal Intensity | සංවේදනාවන්හි උච්චාවචනයන් (Fluctuations) නිරීක්ෂණය. |
| **සඤ්ඤා** | Pattern Recognition | දත්ත ලේබල් කිරීමේ (Labeling) සහ මැකී යාමේ වේගය. |
| **සංඛාර** | Algorithmic Processing | සිතුවිලි සකස් වන රේඛීය නොවන (Non-linear) ස්වභාවය. |
| **විඤ්ඤාණ** | Central Processing Unit | දැනුවත්භාවය නැමැති "Operating System" එකේ පවතින අඛණ්ඩ නොවන ගලායාම. |

#### 2.3 තාරකා විද්‍යාත්මක දත්ත සහ කාල විද්‍යාව (Astro-System Chronology)
System 423 ට අනුව, අප්‍රමාදය යනු විශ්වීය කාල රාමුවක් තුළ දත්ත සැකසීමේ නිරවද්‍යතාවයි. පෞරාණික හෙළ ඉංජිනේරු ශිල්පයේදී සහ නැකත් විද්‍යාවේදී **"ප්‍රමාදය"** යනු පද්ධතිමය ප්‍රලම්භනයකි (**Latency Error**).

* **Data Lag (ප්‍රමාදය):** තාරකා විද්‍යාත්මක දත්ත හෝ ආකාශ වස්තූන්ගෙන් ලැබෙන සංඥා නියමිත නැකත් හෝරාවට වඩා ප්‍රමාද වී (Delayed) සැකසීමෙන් පද්ධතියේ අවසාන ප්‍රතිඵලය (Output) විකෘති වේ.
* **Synchronization Logic:** ධම්මපදය පවසන "අප්‍රමාදය නිවනට මගයි" යන කරුණ, පද්ධති ඉංජිනේරු විද්‍යාවේ **Clock Synchronization** සංකල්පයට සමානය. පද්ධතිය විශ්වීය "තත්‍ය කාලීන" (Real-time) දත්ත සමඟ සමමුහුර්ත වූ විට දෝෂ (Errors) අවම වේ.

#### 2.4 System 423: Watchdog Protocol
ඉංජිනේරු විද්‍යාවේදී **Watchdog Timer** එකක් මගින් පද්ධතියක් "Hang" වීම වළක්වයි. අප්‍රමාදය යනු මිනිස් පද්ධතිය තුළ ක්‍රියාත්මක වන Watchdog Protocol එකයි.

* **ප්‍රමාදය (Negligence):** පද්ධතියේ **Sampling Rate** එක අඩු වීමයි. එවිට දත්ත අතර පවතින හිස්තැන් (Gaps) හඳුනා ගැනීමට නොහැකි වී පද්ධතිය ස්ථිර දෙයක් ලෙස පෙනෙන්නට ගනී (සක්කාය දිට්ඨිය).
* **අප්‍රමාදය (Vigilance):** පද්ධතියේ දත්ත හඳුනාගැනීමේ වේගය (Resolution) උපරිම කිරීමයි. එවිට ස්කන්ධයන්ගේ උදය වැය (හටගැනීම සහ බිඳීම) පැහැදිලිව දර්ශනය වේ.
